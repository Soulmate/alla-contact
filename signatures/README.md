# Email Signatures

This folder contains HTML email signature assets for the contacts in this project.

The signatures are intended to complement the digital business cards stored in the repository root, as well as the `.vcf` contact files and QR codes stored in the sibling folders.

## Structure

- `alla/` - personal email signature files for Alla, including the HTML markup and profile photo.
- `shared/` - reusable icon assets shared by all signature variants.

## Current Contents

The folder now includes signature pages for all listed team members.

Each signature:

- is built as a standalone HTML file,
- includes contact details, company information, and social/web links,
- follows the same visual structure for consistency,
- uses either a real profile image or a text placeholder when no photo asset is available.

## Purpose

These files are meant to be used as a source for branded email signatures. The structure is already suitable for adding more team members by creating one subfolder per person and reusing the shared icons.
